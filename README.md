# Solari Co Linux Rice

## Packages Needed

- **i3-wm** The Window Manager
- **i3status** Status things
- **i3lock-color** Amazing lockscreen
- **polybar** The status bar
- **rofi** App launcher
- **dunst** Notifications
- **lightdm** Log in screen
- **lightdm-slick-greeter** A lightdm theme
- **kitty** Terminal Emulator
- **starship** Make terminal look better
- **insert text editor here** Text editor
- **zsh** zsh > bash
- **timeshift** backups or something
- **playerctl** Media controls
- **ttf-jetbrains-mono-nerd** Nerd Font
- **nordic-theme** Theme or something
- **bibata-cursor-theme** Cursor
- **zafiro-icon-theme** Icons
- **sl** needed 100%

## Setup
Install all the needed packages

Move everything in the .config folder to $HOME/.config

put the .gtkrc-2.0 in your home directory

move everything in the lightdm folder to /etc/lightdm

put your background image in /usr/share/backgrounds/wallpaper.jpg
